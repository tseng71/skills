# cmb-retail-ppt

`cmb-retail-ppt` 用于制作招商银行零售金融风格的中文 16:9 PPT 图片页。

这个 skill 的当前工作方式是：Codex 内置 `image_gen` 只生成每页的**可变内容层**，固定模板元素由用户后续叠加到生成图片上。

## 适合场景

- 招商银行 / 招行 / CMB / 零售金融风格 PPT
- 中文金融、财富管理、资产配置、市场策略类汇报
- 需要每页先生成一张 16:9 图片，再装订成 PPTX
- 固定模板层需要严格保留，不希望生图模型画错 logo、红条、水印或页码

## 当前生成边界

`image_gen` 会生成：

- 封面右侧主标题，最多一个副标题
- 封面级低干扰背景或主题图
- 内页标题、正文、图表、流程图、矩阵、仪表盘、策略框等可变内容

`image_gen` 不生成：

- 封面左侧红色品牌区
- 封面葵花 / 花瓣水印
- 招商银行 / 零售金融品牌锁定
- 封面右侧红色竖条
- 内页右上角 logo 区
- 内页左右红色竖条
- 页码

这些固定模板元素需要用户后续直接叠加。

## 关键规则

- 开始前必须先确认页数。
- 如果用户没有提供附件文档，还必须先确认主题。
- 先生成每 8 页一组的提示词，并等待用户确认。
- 确认后才逐页调用 Codex 内置 `image_gen`。
- 每页只生成一张完整 16:9 图片，不生成拼图或缩略图墙。
- 预留给固定模板层的位置必须完全不可见，不能出现方框、虚线、灰色占位、标签或“预留区域”等文字。
- 封面只允许主标题，最多一个副标题；可以有适合封面的低干扰视觉背景，但不能做成内页式信息图。
- 内页需要图文并茂，适合正式演讲。

## 安装

Codex:

```bash
mkdir -p ~/.codex/skills
cp -R cmb-retail-ppt ~/.codex/skills/cmb-retail-ppt
```

安装后重启 Codex。

## 用法示例

```text
使用招行 PPT skill，根据这个文档做一份 12 页招商银行零售金融风格 PPT。
```

如果没有附件文档：

```text
使用招行 PPT skill，做一份 10 页招商银行零售金融风格 PPT，主题是 2026 年权益市场配置展望。
```
